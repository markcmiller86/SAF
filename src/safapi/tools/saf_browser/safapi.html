SAF Browser

Quick Start Reference

On Subset Lattice and Suite Trees:

  • Single left clicking on a set will print set and collection info to the detail window.
  • Middle clicking on a set shows a popup menu for selecting collection -> field info on that set.
  • Right click on a set will display subset relations dialog if the set is a subset.
  • If a set is a superset in one or more subset relations, double left-clicking on the node representing the set will "open" the superset and display the subsets.

On Topo Tree:

  • If a set is a "target" in a topo relation, double left-clicking on the node representing the set will "open" the set and show the source sets in the topo relation.
  • Right clicking on a source set will show a topo relation data window.
